# Blackhyle
This is our first semester project for CS department. In AICT course, we learnt html,css and javascript and then implemented our knowledge as a group of 4 members to this website.
Blackhyle is basically an ecommerce product showcase website built only on frontend. It showcases different tech related products.
It consists of 5 pages:
- Home page
- Shop Now Page
- About Page
- New Arrivals Page
- Cart Page
